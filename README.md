    # Just Math - Kid Mode

    A math puzzle game for kids aged 7-12 with 4 modes: Divide, Multiply, Add, Subtract.

    ## Setup

    ```bash
    npm install
    npm start
    ```

    ## Game Modes

    - **Divide** – Match tiles where one divides the other evenly
    - **Multiply** – Merge tiles whose product stays under the level cap
    - **Add** – Pair tiles that sum to a target (10, 15, 20)
    - **Subtract** – Merge tiles that are close in value

    ## Level Progression

    - Levels 1-3: 4×4 grid, easy tiles
    - Levels 4-7: 4×4 grid, medium tiles
    - Levels 8-11: 5×5 grid, medium tiles
    - Levels 12+: 5×5 grid, hard tiles
    - Every 4th level: Breather (easy tiles + bonus trash)
